# # 🌱 Farmers & Trustees Trust Platform

A modern, responsive web platform built with **Next.js**, **TypeScript**, and **Tailwind CSS** to empower farmers, support estate planning, connect talent with opportunities, and foster enterprise growth across Zambia.

---

## 📖 Overview

The **Farmers & Trustees Trust (FTT) Platform** is designed to provide a digital ecosystem that supports:

- 🌾 Agriculture & Market Access
- 👨‍👩‍👧 Estate Planning & Family Legacy
- 💼 TalentLink Recruitment Hub
- 📈 Enterprise & Investment Connect
- 🤝 Membership Registration
- 📢 News, Events & Community Engagement

The platform aims to bridge the gap between farmers, investors, professionals, and communities while promoting sustainable development and financial inclusion.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Farmers-trustees-trust-platform.git
```

Navigate into the project

```bash
cd Farmers-trustees-trust-platform
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
